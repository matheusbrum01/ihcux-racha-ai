Protótipo de Sistema (Miro)
Descrição

Este repositório apresenta um protótipo de sistema desenvolvido durante uma atividade em sala de aula, utilizando a ferramenta Miro.

O objetivo do trabalho foi planejar e estruturar a ideia de um sistema, focando na organização visual, fluxo de navegação e experiência do usuário, sem a implementação de código.

Objetivo
Desenvolver a lógica de um sistema de forma visual
Estruturar telas e fluxos de navegação
Trabalhar conceitos de prototipação
Simular a experiência do usuário (UX)
Ferramentas Utilizadas:
Miro
Navegador Web
Git e GitHub para versionamento
Protótipo

O protótipo foi desenvolvido de forma visual no Miro, contendo:

Estrutura de telas
Fluxo de navegação
Organização de funcionalidades
Representação da interface do usuário

(!) Este projeto não possui código executável, pois se trata apenas de um protótipo conceitual.
